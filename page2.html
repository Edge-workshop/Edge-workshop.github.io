WeatherCards

PreviousNextClose

WeatherCards is an application on the iOS platform that gives information on the weather of the day, the next day and the week. All this with the probabilities of precipitation, wind strength, humidity, conversion to Cº and Fº, visibility and atmospheric pressure. There is also the possibility to make a list of its favorite cities to view their weather on one page. The Sketch program, was used to make the design of the application, the geolocation system of the device was also used to locate the user. The programming language used is Swift and CoreData for data backup.

GoodTimes

PreviousNextClose

GoodTimes is an application made for the iOS platform as a final project at school. It is an application that freezes the precious moments in time and groups them in the same place. Every time the user takes a photo, GoodTimes automatically takes GPS coordinates and accurately shows the location on the map where it was taken with the date and time. Moreover if the user desires, he can insert a note or a description to each of the photos. The user also has the option of activating night vision for easier visibility. For this project, FireBaseAuth and FaceBookSDKLoginKit were used for the connection and CoreData for the photo backup as well as MapKit for the geolocation.

Road Test

PreviousNextClose

Road Test was created for the Android platform. This is a quiz that tests the knowledge of rules and driving behavior on our roads. It is a questionnaire with 2 different sections containing 20 questions each, on "road signs" and "road regulations". The player has a time limit to answer each of the questions, failing to answer in time an error is given. At the end of the game, the application counts the points and gives it a result of 100. The result is saved in memory so that the player can go back there in order to beat his own score. All the code is made in Java.
